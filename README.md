# 🍲 MERN Stack Food Recipe App

A full-stack Food Recipe web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).  
Users can explore recipes, save favourites, and manage their own recipe collection.

---

## 🚀 Features

- 🥗 View food recipes
- ❤️ Add to favourites
- 🔍 Search recipes
- 👨‍🍳 Create and manage recipes
- 🔐 User authentication
- 📱 Responsive UI
- ⚡ Fast and interactive frontend

---

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB
